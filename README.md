## how to run

```
npm install // install some modules
npm start // start app
```

## how to manually deploy on netlify

```
npm run build
netlify deploy --dir public // specify directory
netlify deploy --prod
```

alternatively, you can deoloy this app through travis CI.

```
git push
```
